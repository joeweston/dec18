About

This is a series of websites website that I am making, at speed, whilst still making user freindly across multipe devices. I have set self inflicted rules in order to keep my SCSS consistant and reduce the amount of code needed.

Layout Rules

Reset styles using the stylesheet fromm http://meyerweb.com/eric/tools/css/reset/.

Make box sizing border-box

In order of precidence, adjust layout in this order: line-height > padding > margin

Favor using padding-top/margin-top and padding-right/margin-right over other cardinal directions for adjusting layout.

Insights

The most time consuming part of the project was designing the navigation for mobile. This has always been a hard choice but I think flex-wrap and no veriticle lines was a good compromise.

The layout rules made the job quick and SCSS mixins for differing screen sizes kept code to a minimum.

I could have used a different background for mobile screens to keep the same reddish theme - but knowing how to do it is good enough for me

Since I'm making these sites at speed, spelling and grammar can be ignored.
